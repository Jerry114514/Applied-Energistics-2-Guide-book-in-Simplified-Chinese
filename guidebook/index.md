# 应用能源2 是什么?

应用能源2增加了组件和机械，以提供物流和存储解决方案。你可以用AE网络来替代大量的箱子, 但这只是第一步。
应用能源2 旨在与其他模组配合使用，并实现原版游戏或其他模组的自动化。 您可以将系统设置为只需单击一下，
即可制作复杂制作链下的所有必备材料（和最终产物），或保持一定
库存中的物品数量，按需制造，或者只是在基地附近转移物品。
